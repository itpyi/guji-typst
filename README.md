# 古籍雙行夾註排版（Typst）

本項目使用 Typst 實現古籍風格排版，包含：

- 右起豎排網格布局
- 雙行夾註排版規則
- 標點不佔格、右下角紅色擠壓標點
- 一級標題分頁與標題分段規則
- 中縫版心（中間空列）與頁碼/標題裝飾
- 外框與紅色格線樣式

## 文件結構

- `main.typ`：示例正文
- `style.typ`：排版模板與核心邏輯
- `TODO.md`：需求與完成記錄
- `TEMPLATE_FLOW.md`：模板運行機制說明（面向人類閱讀）

## 使用方式

在項目目錄編譯：

```bash
typst compile main.typ
```

輸出文件為 `main.pdf`。

## 字體

模板默認使用 `KingHwa_OldSong`。如本機未安裝，請先安裝或自行替換字體設定。
